# Sharphound Data Collection
SharpHound outputs several JSON files into a Zip file by default and contains the following data in a full enumeration.
-   Local Group Membership
-   AD Group Membership
-   Session Data
-   Domain Trust
-   ACL's
-   AD Objects
-   Containers
